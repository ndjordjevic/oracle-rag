# Oracle-RAG

A powerful PDF RAG (Retrieval-Augmented Generation) system built with LangChain and LangGraph, designed to be wrapped as an MCP (Model Context Protocol) server.

## Overview

Oracle-RAG provides intelligent document querying and retrieval capabilities for PDF documents, leveraging advanced RAG techniques to deliver accurate and contextually relevant answers.
