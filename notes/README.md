# Notes

This folder contains markdown documentation about the learning process, decisions, and insights gathered while building Oracle-RAG.

## Purpose

Document:
- Design decisions and rationale
- Implementation challenges and solutions
- Key learnings from LangChain/LangGraph
- Architecture choices
- MCP server integration notes
- Testing and debugging insights

## Structure

Feel free to organize notes by topic, date, or feature. Examples:
- `rag-implementation.md` - RAG implementation details
- `langgraph-design.md` - LangGraph architecture decisions
- `mcp-integration.md` - MCP server wrapping process
- `vector-store-choices.md` - Vector store selection and configuration
